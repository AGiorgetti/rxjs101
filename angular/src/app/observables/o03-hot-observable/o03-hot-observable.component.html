experiment how we make the observable hot:

  • publish().refCount() - does not work! "late bound" component will not get any value
  • publishLast().refCount() or publishReplay(1).refCount() - will work
  • share() - will partially work: a new http request will be made; also share() has a problem with the initial value: it will be delivered only to the first subscription
  • shareReplay(1) - will work
