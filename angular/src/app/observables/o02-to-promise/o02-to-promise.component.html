Convert the Observable to a Promise: we are down to ONE (1) web request!


The .toPromise() operator subscribes the observable, cretaes and returns a Promise.

The Promise will be resolved when the observable complete.

The Promise will be rejected in case of errors.


It works but it has some drawbacks:

  • the web request will be made up front, event if there's no binding that displays the value.
  • we've lost the power all the RxJS operators that can be used to transform the data.
